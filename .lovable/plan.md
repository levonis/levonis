

## خطة التحسين الشاملة - 7 مهام

هذه الخطة تشمل تحسين الأداء وإعادة تصميم 6 صفحات بشكل احترافي يتناسب مع ثيم الموقع (Taobao-inspired, glassmorphism, RTL).

---

### 1. تحسين الأداء والسرعة

| التغيير | التفاصيل |
|---------|---------|
| Lazy loading للصفحات | تحويل جميع routes لاستخدام `React.lazy()` + `Suspense` |
| تحسين الصور | إضافة `loading="lazy"` و `decoding="async"` لكل `<img>` |
| تقليل imports غير مستخدمة | ProductDetail يستورد 60+ أيقونة - تنظيف ما لا يُستخدم |
| React.memo | تغليف المكونات المتكررة (ProductCard, بطاقات الطلبات) |
| staleTime | إضافة `staleTime` للـ queries المتكررة لتقليل الطلبات |
| Skeleton loading | استبدال Loader2 spinner بـ Skeleton placeholders في كل الصفحات |

**الملفات**: `App.tsx` (lazy routes), `ProductDetail.tsx`, `Favorites.tsx`, `MyOrders.tsx`, `OrderDetail.tsx`, `UserAddresses.tsx`, `ProfileSettings.tsx`

---

### 2. إعادة تصميم صفحة تفاصيل المنتج (`ProductDetail.tsx`)

- **تخطيط جديد**: صور على اليمين بشكل sticky عند التمرير (desktop)، معلومات المنتج على اليسار
- **معرض صور محسّن**: thumbnails أفقية أسفل الصورة الرئيسية بحجم أكبر
- **بطاقة السعر**: تصميم بارز مع خلفية gradient خفيفة وشارة الخصم
- **أقسام منظمة**: كل قسم (خيارات، ألوان، شحن) في card منفصل مع أيقونات واضحة
- **أزرار الإجراء**: شريط ثابت أسفل الشاشة (mobile) مع زر الإضافة للسلة + المفضلة
- **Breadcrumb محسّن**: أصغر وأنظف
- **إزالة backdrop-blur** من الخلفية لتحسين الأداء

---

### 3. إعادة تصميم صفحة المفضلة (`Favorites.tsx`)

- **header مدمج**: أيقونة قلب + عنوان + عداد المنتجات في شريط واحد
- **بطاقات أصغر**: تصميم compact مع صورة مربعة صغيرة + معلومات بجانبها (list view)
- **Swipe to delete**: إضافة زر حذف سريع بتمرير
- **حالة فارغة محسّنة**: رسم توضيحي أجمل مع رسالة تشجيعية
- **Skeleton loading** بدل spinner

---

### 4. إعادة تصميم صفحة العناوين (`UserAddresses.tsx`)

- **header مدمج**: مثل MyOrders - شريط علوي بسيط مع زر إضافة
- **بطاقات عناوين محسّنة**: تصميم rounded-2xl مع أيقونة موقع ملونة، شارة "افتراضي" أوضح
- **أزرار إجراء**: أيقونات فقط في زاوية البطاقة (تعديل/حذف) بشكل أنظف
- **حالة فارغة**: رسم MapPin كبير مع رسالة ودية

---

### 5. إعادة تصميم صفحة طلباتي (`MyOrders.tsx`)

- **تحسينات طفيفة**: الصفحة جيدة أصلاً (Taobao-style) - تحسينات في:
  - تقليل المسافات بين البطاقات
  - إضافة divider خفيف بين الطلبات
  - تحسين أنيميشن hover
  - Skeleton محسّن يطابق شكل البطاقة

---

### 6. إعادة تصميم صفحة الإعدادات (`ProfileSettings.tsx`)

- **تخطيط أقسام واضح**: كل قسم في Card مع عنوان وأيقونة (الملف الشخصي، الأمان، اللغة، الإشعارات)
- **Avatar section**: أكبر مع تأثير ring gradient
- **حقول الإدخال**: تصميم أنظف مع labels أوضح
- **أزرار الإجراء**: شريط حفظ ثابت أسفل الشاشة محسّن مع gradient

---

### 7. إعادة تصميم صفحة تفاصيل الطلب (`OrderDetail.tsx`)

- **header مبسّط**: رقم الطلب + حالة في شريط واحد sticky
- **Timeline محسّن**: أيقونات أصغر مع خط اتصال أرفع
- **بطاقات المعلومات**: تصميم compact مع أيقونات ملونة
- **قسم المنتجات**: بطاقات منتجات بتصميم يشبه MyOrders (صورة صغيرة + معلومات)
- **ملخص الطلب**: تصميم بارز مع خلفية gradient خفيفة للإجمالي
