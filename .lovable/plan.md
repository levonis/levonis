

## تحسين المنصة لتبدو ثلاثية الأبعاد حقيقية

### المشكلة الحالية
المنصة حالياً عبارة عن طبقات مستطيلة مكدسة فوق بعض (top → mid → front → bottom) بدون ميلان حقيقي — تبدو مسطحة وليست ثلاثية الأبعاد. السطح العلوي (`cube-top-featured`) يستخدم `perspective + rotateX` لكن باقي الأجزاء مسطحة تماماً.

### الحل
تحويل المنصة إلى شكل بيضاوي (ellipse) مائل يشبه القاعدة الحقيقية — سطح علوي بيضاوي مائل + جوانب تعطي عمقاً:

```text
        ╭─── المنتج ───╮
        │   فوق المنصة  │
        ╰───────────────╯
      ╭═══════════════════╮   ← سطح بيضاوي مائل (ellipse)
     ╱                     ╲
    ║   السعر + الخصم       ║  ← جسم المنصة (gradient عمودي)
     ╲                     ╱
      ╰═══════════════════╯   ← حافة سفلية + انعكاس
```

### الملف المتأثر
- `src/index.css` فقط — تعديل CSS classes للمنصة

### التعديلات التفصيلية

#### 1. `.cube-top-featured` — سطح بيضاوي مائل
- تغيير من مستطيل إلى `border-radius: 50%` (شكل بيضاوي)
- زيادة `rotateX` إلى ~35-40deg لميلان أوضح يعطي إحساس بالعمق
- تقليل الارتفاع وزيادة العرض ليبدو كسطح منصة حقيقي

#### 2. `.cube-mid-featured` — جسم المنصة
- تحويل إلى شكل بيضاوي مطابق
- تدرج عمودي داكن يعطي إحساس بالجوانب

#### 3. `.cube-front-featured` — الوجه الأمامي مع السعر
- إبقاء مستطيل مع زوايا سفلية مستديرة كبيرة
- أو تحويله لشكل شبه بيضاوي مع clip-path لعمق أفضل

#### 4. `.cube-bottom-edge` + `.cube-bottom-reflection`
- تحويل لبيضاوي مطابق

#### 5. ضبط `marginBottom` للصورة
- التأكد من أن الصورة تبدو وكأنها تقف فوق المنصة المائلة
