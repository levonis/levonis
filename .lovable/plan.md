

## تبسيط المنصة — مستطيل ثلاثي الأبعاد بسيط

### المشكلة
المنصة الحالية بيضاوية بالكامل مع منحنيات مبالغ فيها (`border-radius: 50%`) مما يجعلها تبدو مشوهة وغير طبيعية.

### الحل
استبدال المنصة البيضاوية بمستطيل بسيط ثلاثي الأبعاد — سطح علوي مائل قليلاً + وجه أمامي مسطح:

```text
     ╔══════════════════════╗  ← سطح علوي مائل (perspective rotateX)
     ║░░░░░░░░░░░░░░░░░░░░░░║
     ╠══════════════════════╣  ← حافة علوية مضيئة
     ║                      ║
     ║    السعر + الخصم      ║  ← وجه أمامي مسطح
     ║                      ║
     ╚══════════════════════╝  ← حافة سفلية + انعكاس
```

### الملف المتأثر
- `src/index.css` فقط

### التعديلات

#### 1. `.cube-top-featured`
- إزالة `border-radius: 50%` → استبدالها بـ `border-radius: 8px` (زوايا مستديرة خفيفة)
- إبقاء `perspective + rotateX` بزاوية أقل (~25deg) لإعطاء ميلان بسيط للسطح
- عرض مطابق للوجه الأمامي

#### 2. `.cube-mid-featured`
- إزالة `border-radius: 0 0 50% 50%` → جعله مستطيل عادي بدون منحنيات
- ارتفاع صغير (4-6px) كفاصل بين السطح والوجه

#### 3. `.cube-front-featured`
- إزالة `border-radius: 0 0 50% 50% / 0 0 40% 40%` → استبدالها بـ `border-radius: 0 0 8px 8px`
- إبقاء التدرج اللوني والسعر كما هو

#### 4. `.cube-top-highlight`
- تحويل من بيضاوي إلى مستطيل بزوايا خفيفة

#### 5. `.cube-bottom-edge` + `.cube-bottom-reflection`
- تحويل من بيضاوي إلى مستطيل بزوايا خفيفة

#### 6. Responsive (media query)
- تحديث الأبعاد لتتناسب مع الشكل المستطيل الجديد
