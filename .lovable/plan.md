

# إعادة تصميم صفحة المنتجات (`/products`) بتصميم احترافي مستوحى من الصورة

## الرؤية
تحويل صفحة المنتجات الحالية من تصميم شبكة عادي إلى تصميم فاخر بخلفية خضراء زمردية عميقة، مع قسم Hero لمنتج مميز وبطاقات منتجات بأحجام متفاوتة (Masonry) مع تأثيرات ظلال وإضاءة.

---

## الهيكل الجديد

```text
┌──────────────────────────────────────┐

│                                      │
│  [نص المنتج المميز]  [صورة المنتج   │  ← Hero Section
│   اسم + وصف           بتأثير 3D     │     (أول منتج featured)

```

## التغييرات المطلوبة

### 1. إعادة تصميم `src/pages/Products.tsx`
- **شريط علوي جديد**: شعار الموقع (`/og-logo.png`) على اليسار، حقل البحث على اليمين (RTL)
- **قسم Hero**: جلب أول منتج مميز (`featured = true`) وعرضه بتصميم منقسم:
  - يمين: صورة المنتج بتأثير 3D (transform perspective + rotateY خفيف) مع ظل كبير وتأثير إضاءة radial gradient فوقه
  - يسار: اسم المنتج بخط كبير + الوصف + السعر + زر "عرض المنتج" يوجه لصفحة المنتج
- **شبكة المنتجات (Masonry)**: بطاقات بأحجام متفاوتة (بعضها أطول من الآخر) باستخدام CSS Grid مع `grid-row: span 2` لبعض البطاقات
- **الخلفية**: تدرج أخضر زمردي عميق (`from-[#1a4a2e] via-[#1d5c38] to-[#0f3d22]`)
- إبقاء الفلاتر والترتيب والـ Pagination كما هي لكن بتصميم شفاف يتماشى مع الخلفية الخضراء

### 2. إنشاء `src/components/ProductHeroCard.tsx` (مكون جديد)
- يعرض المنتج المميز بتأثير 3D
- صورة المنتج مع:
  - `drop-shadow` كبير للأسفل
  - Radial gradient شفاف فوقها لتأثير الإضاءة
  - `transform: perspective(800px) rotateY(-5deg)` لتأثير ثلاثي الأبعاد خفيف
- النص والوصف وزر CTA على الجانب الآخر

### 3. إنشاء `src/components/ProductMasonryCard.tsx` (مكون جديد)
- بطاقة المنتج الجديدة بتصميم يطابق الصورة:
  - خلفية البطاقة بلون أخضر أفتح شفاف (`bg-white/10 backdrop-blur`)
  - زوايا دائرية كبيرة (`rounded-2xl`)
  - صورة المنتج تبرز فوق البطاقة مع ظل (`drop-shadow-2xl`)
  - اسم المنتج والسعر أسفل الصورة
  - عند الضغط: التوجه لصفحة المنتج (`/product/{slug}`)
- بعض البطاقات أكبر من الأخرى (تناوب بين `aspect-[3/4]` و `aspect-[3/5]`)

### 4. إضافة خيار اختيار "منتج Hero" في صفحة الإدارة
- في `src/pages/Admin.tsx`: إضافة حقل checkbox جديد `is_hero` أو استخدام الحقل الموجود `featured` لتحديد المنتج الذي يظهر في قسم Hero
- سيتم استخدام أول منتج `featured = true` كمنتج Hero تلقائياً (بدون حاجة لعمود جديد في DB)

## التفاصيل التقنية

- **لا حاجة لتغيير DB**: سنستخدم حقل `featured` الموجود لتحديد منتج Hero (أول واحد)
- **الألوان**: خلفية خضراء داكنة مع بطاقات `bg-white/10` وخطوط بيضاء/ذهبية
- **RTL**: التصميم يدعم RTL بالكامل (البحث يمين، الشعار يسار)
- **الأداء**: Masonry بـ CSS Grid فقط (بدون مكتبات إضافية)
- **التوافق**: الفلاتر والبحث والترتيب والـ Pagination تبقى تعمل كما هي
