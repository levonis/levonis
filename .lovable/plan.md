

# إعادة تصميم آلة الغاتشا لتطابق الصورة المرجعية

## الفرق بين التصميم الحالي والمطلوب

التصميم الحالي: زجاجة كروية (globe) + جسم معدني داكن + مقبض دائري في المنتصف
التصميم المطلوب (من الرابط): آلة مستطيلة بأسلوب رسوم توضيحية (illustration) مع:

```text
         🔴  ← زر أحمر دائري أعلى الإطار
    ┌──────────────┐
    │ ╔══════════╗ │  ← إطار ملون (وردي/أحمر)
    │ ║ ⚪⚪⚪⚪ ║ │  ← نافذة زجاجية مستطيلة مع كبسولات كبيرة
    │ ║ ⚪⚪⚪⚪ ║ │     متراصة بشكل عشوائي
    │ ║ ⚪⚪⚪⚪ ║ │
    │ ╚══════════╝ │
    │●●●●●●●●●●●●●│  ← شريط زخرفي من نقاط ذهبية
    │  ⊕     ━━━  │  ← مقبض دائري (يسار) + فتحة خروج (يمين)
    │ ▐████▌       │  ← فتحة استلام مستطيلة (أسفل يسار)
    └──────────────┘
    ═══════════════  ← قاعدة ذهبية
```

## التغييرات

### 1) `GachaMachineVisual.tsx` — إعادة رسم كامل
- نافذة زجاجية **مستطيلة** بزوايا مدورة (ليست كروية)
- كبسولات **كبيرة** مع خطوط تقسيم واضحة وألوان باستيل (وردي، أزرق فاتح، أبيض، أصفر فاتح، أخضر فاتح)
- إطار ملون بحدود خارجية واضحة (ليس تدرج معدني)
- شريط نقاط زخرفي أفقي بين النافذة والجسم السفلي
- مقبض دائري على **يمين** الجسم السفلي (ليس في المنتصف) بخط أفقي
- فتحة خروج صغيرة مستطيلة أسفل يسار الجسم
- زر دائري أحمر/ملون أعلى الآلة
- قاعدة ذهبية/ملونة أسفل
- أسلوب flat illustration بحدود واضحة (outline stroke)
- ألوان الثيم تُطبق على الإطار والجسم

### 2) `GachaSpinReveal.tsx` — لا تغييرات جوهرية
الأنيميشن يبقى كما هو (نزول الكرة + فتح لنصفين)

### 3) `GachaMachineCard.tsx` و `GachaLanding.tsx` — تحديث تلقائي
يستخدمون `GachaMachineVisual` فيتحدثون تلقائياً
