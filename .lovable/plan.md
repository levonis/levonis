

## لعبة البنك - استثمار محفوف بالمخاطر 🏦

لعبة pixel art جديدة: يستثمر اللاعب نقاطه في "البنك" ويختار مدة زمنية. كلما زادت المدة زادت الفائدة وزاد خطر السرقة.

---

## آلية اللعبة

```text
┌──────────────────────────────────────────────────┐
│                 🏦 PIXEL BANK                     │
│                                                   │
│  ① اختر المبلغ: [slider أو input] من نقاطك       │
│                                                   │
│  ② اختر المدة:                                    │
│  ┌───────┬────────┬────────┬────────┬───────────┐  │
│  │ أسبوع │أسبوعين │ شهر   │6 أشهر │ سنة       │  │
│  │ +1%   │ +4%    │ +10%  │+100%  │ +500%     │  │
│  │ خطر   │ خطر   │ خطر   │ خطر   │ خطر       │  │
│  │ 5%    │ 15%   │ 25%   │ 50%   │ 75%       │  │
│  └───────┴────────┴────────┴────────┴───────────┘  │
│                                                   │
│  ③ اضغط "استثمر" → رسوم متحركة خزنة              │
│  ④ محاكاة فورية: هل سُرق البنك؟                   │
│     ✅ نجاح → تحصل على المبلغ + الفائدة           │
│     🦹 سرقة جزئية → خسارة 20-80% من المبلغ        │
│     💀 سرقة كاملة → خسارة كل المبلغ               │
└──────────────────────────────────────────────────┘
```

**نسب الخطر:**
| المدة | الفائدة | احتمال النجاح | سرقة جزئية | سرقة كاملة |
|-------|---------|--------------|------------|------------|
| أسبوع | 1% | 85% | 12% | 3% |
| أسبوعين | 4% | 70% | 22% | 8% |
| شهر | 10% | 55% | 30% | 15% |
| 6 أشهر | 100% | 35% | 35% | 30% |
| سنة | 500% | 15% | 35% | 50% |

**اللعبة فورية** - لا انتظار حقيقي. اللاعب يختار المدة ويضغط "استثمر" فتظهر رسوم متحركة للخزنة ثم النتيجة فوراً.

---

## التفاصيل التقنية

### 1. ملف جديد: `src/components/games/BankGame.tsx`
- يتبع نفس هيكل `RockPaperScissorsGame.tsx` (props: `onBack`)
- رسم الخزنة بـ Canvas pixel art (مشابه لـ PixelCanvas)
- مراحل اللعبة: `setup` → `investing` (animation) → `result`
- استخدام `admin_adjust_points` RPC لإضافة/خصم النقاط
- حد أدنى للاستثمار: 10 نقاط
- حد أقصى: 500 نقطة (أو كل نقاط اللاعب أيهما أقل)

### 2. تعديل `GamesData.ts`
- تغيير بيانات لعبة `"bank"` (تستبدل واحدة من الألعاب القادمة أو تُضاف) بـ status: LIVE

### 3. تعديل `MiniGames.tsx`
- إضافة lazy import لـ `BankGame`
- إضافة شرط `activeGame === 'bank'` لعرض اللعبة

### 4. التصميم (Pixel Art)
- خزنة بنك مرسومة بـ Canvas (16x16 grid مكبّر)
- رسوم متحركة: الخزنة تفتح، عملات تطير، أو لص يظهر
- ألوان ذهبية/خضراء للنجاح، حمراء للسرقة
- خلفية floating pixels مع أيقونات عملات

### الملفات المتأثرة

| ملف | التعديل |
|-----|---------|
| `src/components/games/BankGame.tsx` | **جديد** - اللعبة كاملة |
| `src/components/games/GamesData.ts` | إضافة/تحديث node البنك |
| `src/pages/MiniGames.tsx` | lazy import + routing |

لا حاجة لتغييرات في قاعدة البيانات - يستخدم `admin_adjust_points` الموجود.
