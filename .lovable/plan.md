## الهدف
1. جعل التقييمات تُنشر تلقائياً (approved افتراضياً) بدلاً من انتظار مراجعة الأدمن، مع إبقاء صلاحية الأدمن للحذف أو تعديل التعليق السيء.
2. إعادة تصميم صفحة "تم التسليم" (`ConfirmDelivery`) بواجهة عصرية على غرار Taobao — بسيطة، بصرية، سهلة، مع إمكانية تعديل التقييم التلقائي لاحقاً.

---

## 1) نظام التقييم التلقائي بدون مراجعة

### تغييرات قاعدة البيانات (migration)
- تغيير قيمة `reviews.status` الافتراضية إلى `'approved'`.
- تحديث جميع الصفوف الحالية `status = 'pending'` → `'approved'` (لكي لا تبقى معلّقة).
- إبقاء عمود `status` كما هو حتى لا نكسر الكود، لكن لن يُستخدم للحجب.
- استخدام دالة `award_review_points` (أو تكرار منطق النقاط) داخل trigger `AFTER INSERT` على `reviews` لمنح النقاط فوراً عند التقييم:
  - أساس: 10 نقاط لتقييم 5 نجوم خلال 24 ساعة من التسليم.
  - +50 لكل صورة، +250 للفيديو (نفس المضاعفات الحالية).
  - تحديث `points_awarded = true` لمنع التكرار عند التعديل.

### تغييرات الواجهة
- **`src/pages/ConfirmDelivery.tsx`**: تغيير `status: 'pending'` → `status: 'approved'` في insert وupdate.
- **`src/pages/AdminReviews.tsx`**:
  - إزالة أزرار "قبول/رفض" و تبويب "قيد المراجعة".
  - إبقاء عرض كل التقييمات (سجل موحّد) مع:
    - زر **حذف التقييم بالكامل**.
    - زر **حذف/تعديل نص التعليق فقط** (مسح الـ `review_text` وإبقاء النجوم).
    - فلاتر (تقييم منخفض، يحتوي كلمات مسيئة، بواسطة مستخدم…).
  - إشارة `is_auto_rating` (تلقائي / يدوي).

### تعديل التقييم التلقائي من قبل المستخدم
- عرض التقييمات التلقائية (auto 5★) في صفحة "تم التسليم" وفي `MyPurchasedProducts` مع زر **"عدّل تقييمك"** يفتح نفس نموذج التقييم، ويحوّل `is_auto_rating = false` عند الحفظ.

## 2) إعادة تصميم صفحة "تم التسليم" بأسلوب Taobao

### الرؤية البصرية
- خلفية متدرجة ناعمة + بطاقات زجاجية (glassmorphism) متطابقة مع نظام التصميم.
- Header علوي احتفالي: أيقونة تسليم + رسالة شكر + شارة نقاط متوقعة.
- شريط تقدّم أفقي "قيّم منتجاتك (2/4)" أعلى الصفحة.

### هيكل الصفحة (Mobile-first مثل Taobao)
```text
┌─────────────────────────────┐
│ ✅ تم توصيل طلبك             │
│ رقم #1234 · 4 منتجات          │
│ ▓▓▓▓░░░░ 2/4 مُقيّمة   [+10pts]│
├─────────────────────────────┤
│ [صورة]  اسم المنتج            │
│         ★★★★★ (اضغط للتقييم)  │
│         [+ صور] [+ فيديو]    │
│         [حقل تعليق مطوي]     │
├─────────────────────────────┤
│           …المنتج التالي…    │
├─────────────────────────────┤
│ [تخطي وتأكيد التسليم]        │
│ [إرسال التقييمات وتأكيد]     │
└─────────────────────────────┘
```

### مكونات جديدة
- **`src/components/delivery/DeliveryHeader.tsx`**: بطاقة الترحيب + تقدّم التقييم + إجمالي النقاط المتوقعة.
- **`src/components/delivery/ProductReviewCard.tsx`**: بطاقة منتج واحد بتفاعلات ناعمة:
  - نجوم كبيرة نابضة مع haptic-like animation.
  - أزرار صور/فيديو أفقية مع معاينات مصغرة قابلة للحذف.
  - Textarea تظهر تلقائياً بعد اختيار نجمة.
  - Badge نقاط ديناميكي يتحدث فورياً حسب المرفقات (+50/+250).
- **`src/components/delivery/StickySubmitBar.tsx`**: شريط سفلي ثابت مع CTA رئيسي + خيار "تخطي".

### تحسينات UX
- تحديد كل النجوم 5 افتراضياً (سلوك Taobao) مع إمكانية التغيير.
- عند رفع صورة/فيديو، شارة النقاط تتحرك بأنيميشن.
- Toast احتفالي عند الإرسال + عرض النقاط الممنوحة فعلياً.
- زر عائم "تخطي الكل" يؤكد التسليم بتقييم تلقائي 5★.
- بعد الإرسال: شاشة نجاح مع CTA "متابعة التسوق" و"مشاهدة تقييماتك".

### تعديل التقييم لاحقاً
- في نفس الصفحة، إذا كان الطلب مؤكداً مسبقاً والتقييم تلقائي، تظهر البطاقات بحالة "تلقائي — يمكن التعديل" مع زر واحد لفتح النموذج.
