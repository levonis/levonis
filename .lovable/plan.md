

# تصميم آلة الغاتشا اليابانية مع أنيميشن كامل

## الوصف

إعادة تصميم آلة الغاتشا لتطابق الصورة المرفقة — آلة كبسولات يابانية كلاسيكية بجسم معدني، زجاجة شفافة مليئة بالكرات، مقبض دوّار، وفتحة لخروج الكرة. الألوان تتغير حسب ثيم الآلة (كوبونات = أخضر، دُمى = وردي، إلخ).

### أنيميشن اللف (3 مراحل)
1. **دوران المقبض** — المقبض يدور 360° مع صوت كلك
2. **نزول الكرة** — كرة ملونة تنزلق من بين الكرات في الزجاجة إلى فتحة الخروج أسفل الآلة
3. **فتح الكرة** — الكرة تنقسم لنصفين وتظهر الجائزة بينهما مع إضاءة الندرة

---

## التغييرات التقنية

### 1) `GachaMachineDetail.tsx` — إعادة بناء واجهة الآلة

بدل الإيموجي والصورة الحالية، رسم آلة CSS كاملة تتضمن:

```text
┌──────────────┐  ← غطاء أسود
│ ╔══════════╗ │
│ ║  كرات    ║ │  ← زجاجة شفافة + كرات ملونة متحركة
│ ║ ⚪⚪⚪   ║ │
│ ╚══════════╝ │
├──────────────┤  ← جسم الآلة (لون حسب الثيم)
│  ⚙  ◉       │  ← مقبض + فتحة خروج
│  [اسم]      │  ← لوحة الاسم
└══════════════┘  ← قاعدة
```

- الزجاجة تحتوي 8-12 كرة CSS بألوان الندرة (رمادي، أخضر، أزرق، بنفسجي، ذهبي)
- المقبض قابل للنقر ويدور عند اللف
- فتحة الخروج دائرية سوداء

ألوان الجسم حسب الثيم:
- `default` → أحمر (مثل الصورة)
- `coupon` → أخضر
- `doll` → وردي
- `event` → برتقالي
- `premium` → ذهبي

### 2) `GachaSpinReveal.tsx` — أنيميشن الكرة والفتح

إعادة بناء كامل لتسلسل الكشف:

**المرحلة 1 (0-0.8s):** المقبض يدور مع اهتزاز خفيف للآلة
**المرحلة 2 (0.8-2s):** كرة تنزل من الزجاجة عبر الجسم إلى فتحة الخروج (مسار منحني)
**المرحلة 3 (2-2.5s):** الكرة تتوسط الشاشة وتكبر
**المرحلة 4 (2.5-3.5s):** الكرة تنقسم — النصف العلوي يطلع لفوق والسفلي ينزل، الجائزة تظهر بالمنتصف مع glow بلون الندرة

كل هذا باستخدام `framer-motion` الموجود بالفعل.

### 3) `GachaMachineCard.tsx` — بطاقة الآلة في الشبكة

تحديث البطاقة لتظهر شكل الآلة المصغر بدل الإيموجي:
- آلة صغيرة CSS بنفس التصميم
- الألوان حسب الثيم
- كرات صغيرة داخل الزجاجة

### 4) `GachaLanding.tsx` — تحديث البانر

تغيير البانر ليتناسب مع التصميم الجديد بدل 🎰 emoji.
