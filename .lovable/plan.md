

# إضافة إمكانية حذف وتعديل المسودات المحولة ("تم التحويل")

## المشكلة
المسودات التي تم تحويلها لشحنة (status = 'converted') تظهر بشفافية منخفضة ولا يمكن تعديلها أو حذفها لأن أزرار التعديل والحذف مخفية بشرط `!isConverted`.

## الحل

### ملف: `src/pages/AdminInventory.tsx`

1. **إضافة زر حذف للمسودات المحولة**: نقل زر الحذف (سلة المهملات) خارج شرط `!isConverted` ليظهر دائماً، سواء كانت المسودة مسودة أو محولة.

2. **إضافة زر تعديل للمسودات المحولة** (اختياري): إذا أردت السماح بتعديل المسودة المحولة أيضاً، سيتم نقل زر التعديل خارج الشرط كذلك.

3. **إبقاء زر "تحويل لشحنة" مخفياً** للمسودات المحولة فقط، لأنها حُولت بالفعل.

**التغيير**: في السطور 1165-1190، إعادة ترتيب الأزرار بحيث يبقى "تحويل لشحنة" داخل `!isConverted` بينما يظهر "تعديل" و"حذف" دائماً.
