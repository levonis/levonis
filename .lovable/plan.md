

## خطة إعادة التصميم الجذري - 5 صفحات + شريط التنقل

المشكلة: التغييرات السابقة كانت طفيفة جداً ولم تُحدث فرقاً ملموساً. هذه المرة سيتم إعادة كتابة كل صفحة بالكامل بتصميم مختلف تماماً.

---

### 1. شريط التنقل السفلي (`AppNavBar.tsx`) - إعادة تصميم كامل
- **تقليل العناصر**: 5 عناصر فقط بدل 7 (Home, Cart, Community, Rewards, Profile) - نقل Games و Chat داخل الصفحات
- **تصميم جديد**: شريط أنحف مع أيقونات أكبر، العنصر النشط يظهر كـ pill ملون مرتفع فوق الشريط
- **انيميشن**: تأثير spring عند الضغط مع تغيير سلس للأيقونة النشطة
- **Desktop**: إخفاء الشريط واستخدام sidebar مدمج في الصفحة

### 2. صفحة طلباتي (`MyOrders.tsx`) - إعادة كتابة كاملة
- **Header جديد**: خلفية gradient مع عنوان كبير وعداد الطلبات كرقم بارز
- **Tabs محسّنة**: chips أفقية بتصميم pill مع أيقونات ملونة لكل حالة، العداد داخل الـ chip
- **بطاقات الطلبات**: تصميم جديد كلياً:
  - صورة المنتج كخلفية شفافة خلف البطاقة
  - شريط حالة ملون على يمين البطاقة (vertical accent bar)
  - السعر بخط كبير بارز مع أيقونة العملة
  - تاريخ الطلب في الأسفل بتنسيق نسبي ("منذ 3 أيام")
  - زر التتبع كشريط كامل العرض ملون

### 3. صفحة المفضلة (`Favorites.tsx`) - إعادة كتابة كاملة
- **Header**: خلفية متدرجة وردية/حمراء خفيفة مع أيقونة قلب متحرك
- **Grid view**: عرض المنتجات كـ grid (2 أعمدة) بدل list view
  - كل بطاقة: صورة المنتج تأخذ 70% من البطاقة
  - زر القلب الأحمر في الزاوية العلوية
  - اسم المنتج والسعر في الأسفل
  - زر "أضف للسلة" كشريط أسفل البطاقة
- **حالة فارغة**: رسوم متحركة لقلب نابض مع رسالة

### 4. صفحة تفاصيل المنتج (`ProductDetail.tsx`) - إعادة هيكلة
- **Mobile-first**: الصور تأخذ كامل عرض الشاشة بدون padding/border
- **Image counter**: نقط الصور أسفل الصورة مباشرة بتصميم بسيط
- **معلومات المنتج**: بطاقة مرتفعة تتراكب مع الصورة (overlap) بـ negative margin
- **السعر**: خلفية gradient بارزة مع شارة الخصم بجانبه
- **الأقسام**: accordion بدل cards متتالية لتقليل الطول
- **Sticky bottom bar**: شريط ثابت أسفل الشاشة يحتوي: سعر + زر سلة + زر مفضلة
- **إزالة backdrop-blur** من كل مكان
- **تنظيف الـ imports**: إزالة 50+ أيقونة غير مستخدمة واستبدالها بـ dynamic icon map

### 5. صفحة الإعدادات (`ProfileSettings.tsx`) - إعادة تنظيم
- **Header**: صورة البروفايل كبيرة في المنتصف مع اسم المستخدم تحتها
- **أقسام منفصلة واضحة**:
  - قسم "المعلومات الشخصية" (الاسم، اليوزرنيم، الهاتف) 
  - قسم "العنوان الافتراضي"
  - قسم "الأمان" (PIN)
  - قسم "اللغة"
  - قسم "الإشعارات"
- كل قسم بـ header ملون مع أيقونة وعنوان
- **زر الحفظ**: ثابت أسفل الشاشة مع gradient وأيقونة

### 6. أداء عام
- إضافة `staleTime: 5 * 60 * 1000` لكل queries المنتجات والطلبات
- `loading="lazy" decoding="async"` لكل الصور
- إزالة `backdrop-blur` من كل الصفحات
